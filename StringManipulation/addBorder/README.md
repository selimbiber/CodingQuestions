# addBorder

## Question Description

Given a rectangular matrix of characters, add a border of asterisks(\*) to it.

## Example of Expected Result

```
For
picture = ["abc", "ded"]

the output should be
solution(picture) = ["*****", "*abc*", "*ded*", "*****"]

```
