# Do you speak "English"?

## Question Description

Given a string of arbitrary length with any ASCII characters. Write a function to determine whether the string contains the whole word "English".

The order of characters is important -- the string "abcEnglishdef" is correct but "abcnEglishsef" is not correct.

Upper or lower case letters does not matter -- "eNglisH" is also correct.

Return value as boolean values, true for the string to contain "English", false for it does not.
