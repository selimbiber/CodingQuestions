# Get Nth Even Number

## Question Description

Return the Nth Even Number

The input will not be 0.

## Examples of Expected Results

```js
1 --> 0 (the first even number is 0)
3 --> 4 (the 3rd even number is 4 (0, 2, 4))
100 --> 198
1298734 --> 2597466
```
